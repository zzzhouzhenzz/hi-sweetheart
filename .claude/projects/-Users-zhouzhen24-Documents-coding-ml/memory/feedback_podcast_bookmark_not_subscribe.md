Podcast action = BOOKMARK only. Never subscribe/follow.

**Why:** User has corrected this multiple times. "I DON'T WANT SUBSCRIBE. I just want bookmark." Subscribe and bookmark are different actions in Apple Podcasts — bookmark saves a show/episode for later, subscribe/follow adds it to your library with auto-download.

**How to apply:** When implementing podcast features, always target the bookmark/save action, not follow/subscribe. Don't conflate the two. Don't suggest subscribe as an alternative.
